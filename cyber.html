Cybersecurity

Protect systems, networks, and data from digital threats.

Step 2

Linux & Scripting

Get comfortable with Linux CLI and learn Python/Bash for security scripting.

⏱ 3-4 weeks    📘 OverTheWire · Linux Basics
Step 3

Web Security

Learn about OWASP Top 10, XSS, SQL injection, CSRF, and how to prevent them.

⏱ 3-4 weeks    📘 OWASP · PortSwigger Academy
Step 4

Cryptography

Understand encryption, hashing, digital signatures, and PKI.

⏱ 2-3 weeks    📘 Crypto101 · Khan Academy

Penetration Testing

Practice ethical hacking with tools like Burp Suite, Nmap, and Metasploit.

⏱ 4-6 weeks    📘 TryHackMe · Hack The Box

Security Operations

Learn SIEM, incident response, threat hunting, and security compliance.

⏱ 4-6    📘 Splunk Training · SANS Resources
